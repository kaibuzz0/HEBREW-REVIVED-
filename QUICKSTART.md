# Quick Start Guide

## 3 Steps to Get Started

### Step 1: Launch
```bash
# Linux/macOS
python3 launch.py

# Windows
python launch.py
```

### Step 2: Use the Menu
Choose from:
- 🔍 Search the Archive
- 📊 View Statistics
- 📖 Open Documentation

### Step 3: Explore
- Open `COMPLETE_BIBLE_ARCHIVE.html` in browser
- Run `python3 dashboard.py` for stats
- Check `exports/` for data files

## Common Tasks

**Search:**
```bash
python3 tools/search_system.py
```

**View Dashboard:**
```bash
python3 dashboard.py
```

**Validate System:**
```bash
python3 validate.py
```

## Need Help?

1. Check `README.md` for full documentation
2. Run `python3 validate.py` to check system
3. View `docs/` for additional guides

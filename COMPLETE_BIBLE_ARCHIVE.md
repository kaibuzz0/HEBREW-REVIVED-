## Usage

### View Online
Open `COMPLETE_BIBLE_ARCHIVE.html` in any web browser for the full interactive presentation with statistics, visualizations, and navigation.

### Search the Database
```bash
python3 search_system.py
```

### Run Analysis
```bash
python3 seeds_decoder_v3.py
```

### Browse Database
```bash
sqlite3 complete_bible.db
.database
.tables
SELECT * FROM genesis LIMIT 5;
```
